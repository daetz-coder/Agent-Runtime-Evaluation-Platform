\datedsubsection{\textbf{Wiki-Agent：AI 驱动的智能知识库系统} \quad \textit{LangGraph / FastAPI / Vue3 / Milvus}}{2025.5 - 至今}
\begin{itemize}[parsep=0.15ex,topsep=0ex]
  \item \textbf{设计 Query 改写 Pipeline，解决用户意图模糊导致检索命中率低的问题。}前置路由分类器（LLM 轻量 4 分类），多轮对话指代消解（代词检测 + 上下文补齐），按类型路由到不同改写策略（Multi-Query / HyDE / Decompose）；改写后 cosine 相似度校验（阈值 0.7），低于阈值回退原始 query，防止语义漂移
  \item \textbf{四级混合检索，单路检索召回率不足的瓶颈。}Milvus 语义搜索（bge-small-zh-v1.5 + cosine）+ BM25 关键词搜索（jieba 分词）→ RRF 倒数秩融合（k=60）→ Cross-Encoder 重排（bge-reranker-base），Top-1=\textbf{75\%} / MRR=\textbf{0.825}（20 条查询基准）；多 query 并行检索 + 文档级去重合并
  \item \textbf{4-Node StateGraph 编排检索全流程，实现可控的人机协作。}search $\rightarrow$ respond $\rightarrow$ decide $\rightarrow$ execute，条件路由 + LangGraph interrupt 实现 Human-in-the-Loop CRUD，Checkpoint 恢复；Milvus + BM25 + Git 实时同步，commit 级回滚 + 一键全量重建，Vue3 向量管理后台
\end{itemize}
