
# 一、岗位职责

1. 负责企业级 AI Agent 基础设施平台核心模块开发，包括 MCP 中心、Skill 中心、Agent 中心、AI 网关等系统设计与实现。

2. 负责 Agent 全生命周期管理平台迭代演进，推动 Multi-Agent（A2A）、Agent 运行沙箱、LLM-as-a-Judge 评测体系等方向工程化落地。

3. 持续跟踪 Agent 工程、RAG、LLM 应用开发框架等前沿技术发展，并推动产品化落地。

4. 负责平台基础设施研发与演进，包括数据存储与微服务架构设计与实现。

---

# 二、岗位要求

## 基本要求

1. 全日制本科及以上学历，计算机科学与技术或相关专业；有分布式系统或 AI 工程背景优先。

2. 扎实计算机基础，熟悉算法与数据结构；理解计算机网络、数据库、分布式系统原理。

3. 至少熟练掌握一门后端开发语言（Java / Python）。

4. 具备较强系统设计能力，能够独立完成模块拆解、接口设计到上线的完整研发闭环。

## AI 工程方向（核心加分项）

5. 熟悉 LLM 工作原理与 Agent 工程（ReAct、Tool Calling、Human-in-the-Loop、MCP 协议等），有实际 Agent 应用开发经验；  
   熟悉 LangChain、LangGraph、Spring AI / Alibaba 等框架或动态服务治理相关经验优先。

6. 有 RAG / 向量检索 / 混合检索系统实践经验；  
   熟悉向量数据库、Embedding、BM25 等检索技术。

7. 有微调（Fine-tuning）、Prompt Engineering、LLM-as-a-Judge 等 AI 工程实践经验。

---

## 系统工程方向（加分项）

8. 有分布式系统实战经验；熟悉 Redis、消息队列、分布式锁等常见架构模式；  
   熟悉配置中心（Nacos）、灰度发布、服务治理相关经验优先；  
   有 MySQL 等关系型数据库设计与调优经验优先。

> 以上加分项不要求全部具备，鼓励有深度技术积累者投递。
